# Caffeine Calculator

![Caffeine Calculator Hero Image](./public/hero.png) <!-- Update this path when you have a screenshot -->

Welcome to the **Caffeine Calculator**, a scientifically-backed tool designed to help you determine the caffeine content of your home-brewed cup of coffee! By factoring in the bean species, roast level, brew method, and coffee weight, this Next.js app provides an accurate and visually pleasing breakdown of what’s in your morning brew.

---

## 🚀 Features

- **Interactive Calculator**: A sleek, user-friendly form to input your coffee characteristics.
- **Accurate Estimations**: Factoring in variables like Robusta vs. Arabica, Light vs. Dark roasts, and specific extraction yields from various brew methods.
- **Elegant UI/UX**: Designed with beautiful styling using Tailwind CSS and Radix UI components for an accessible and visually rich experience.
- **Micro-Animations**: Providing immediate feedback to user interactions with smooth transitions.

## 💻 Getting Started

Want to run the Caffeine Calculator locally? Follow these steps!

### Prerequisites
Make sure you have Node.js and npm (or yarn/pnpm/bun) installed.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/zacharywright/caffeine-calculator.git
   ```

2. **Navigate into the project directory:**
   ```bash
   cd caffeine-calculator
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to see the application in action!

## ☕ The Science Behind It
When calculating caffeine yield, we consider four major factors:

1. **Bean Species**: Robusta beans typically have *twice* the caffeine content of Arabica beans.
2. **Roast Level**: Contrary to popular belief, light roasts generally retain slightly more caffeine by volume than dark roasts because they are denser.
3. **Brew Method**: Espresso relies on pressure and yields a very concentrated dose, while methods like French Press or Pour Over have longer extraction times affecting overall yield.
4. **Coffee Weight**: The baseline variable setting up the total possible extracted caffeine.

*Note: The results are scientific estimates based on well-established industry averages. Exact caffeine profiles can vary depending on crop origin and specific extraction variables!*
